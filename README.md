# Student Record Management System

## 📌 Description
Tkinter-based student database app to store, update, search, and delete student records.

## 🚀 Features
- Add new student
- View all records
- Update student info
- Delete student
- Search by roll number

## ⚙️ Technologies
- Python 3
- Tkinter
- SQLite3
- Git + GitHub
